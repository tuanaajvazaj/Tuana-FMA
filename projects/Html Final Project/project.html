Project 3 - Register and Log In Form

Firstly I used the Form Element than I made a label for email and an input type email. Than I also made a label for password and an input tybe password. After that I made an input type submit. After that I added a label for birthday and an input type date. Than I added 2 input type radio for the gender. After this I added an input type checkbox. And lastly, if you already had an account, I added a a-href link that if you click on it it sends you to the first file-login.html.
